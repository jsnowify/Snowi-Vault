# ❄️ Snowi Vault

A personal vault of coding problems solved across various platforms — LeetCode, Codewars, HackerRank, and more. This repo tracks my problem-solving journey, the patterns I've learned, and my progress over time.

## 📌 Purpose

- Practice and reinforce data structures & algorithms
- Keep a searchable archive of solved problems
- Track consistency and growth over time

## 📁 Structure

```
├── leetcode/
│   ├── easy/
│   ├── medium/
│   └── hard/
├── codewars/
│   ├── 8kyu/
│   ├── 7kyu/
│   └── ...
├── hackerrank/
├── others/
└── README.md
```

Each solution folder includes:
- The problem statement (or a link to it)
- My solution file
- Notes on approach / complexity (when useful)

## 🛠️ Languages Used

- Python
- JavaScript/TypeScript
- Java

## 📝 Notes

Recurring patterns, templates, and cheat sheets (sliding window, two pointers, DP, etc.) live in the `notes/` folder for quick reference.

## 🎯 Goals

- [ ] Solve at least 7 problems per week
- [ ] Cover all major DSA topics
- [ ] Revisit and optimize old solutions

⭐ This vault grows one problem at a time.
